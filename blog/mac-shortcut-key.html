Macで使えるショートカットキーの覚え書き

2018-05-08



この記事は随時修正、更新されます。

Finderで使えるショートカットキー

ファイルを複製する

command + D
DはDuplicateの意味

名前の変更

ファイルやフォルダを選択して return

QuickLook

ファイルを選択してスペースキー

新規タブの作成

command + T

タブを閉じる

command + W

親フォルダに移動

command + ↑

選択したファイルを開く

command + O


文字入力で使えるショートカットキー

言語を切り替える

Command + スペース

絵文字を入力する

Control + Command + スペース

参考:
https://www.danshihack.com/2016/05/21/junp/mac-keyboard-emoji.html


スクリーンショットで使えるショートカットキー

command + shift + 4
で全画面スクリーンショット

カーソルが十字になっている状態でスペースを押すとウィンドウだけキャプチャしてくれる。
