Log.dのメッセージ出力には文字数制限あり

2018-11-08

ログ出力のこれ。

Log.d(tag, message)

messageを出力できる最大文字数が設定されているようだ。4000文字くらいで途切れてしまうが、知らずに数時間ハマった。


val list = message.split(Regex("\n"))

list.forEach {
     Log.d('bugfix', it)
}


こんな感じで改行区切りで一行づつ出力ことで、簡易対処することにした。
