TeMaKi ver0.8リリース

2018-12-24

TeMaKiはPythonで作られた静的サイトジェネレーターです

https://github.com/araemon/temaki

MarkDownファイルからHTML生成して、静的サイトコンテンンツを生成できるようにPythonでプログラミングしました。
世の中にはHugoをはじめ、すでに多くの静的サイトジェネレーターが存在しますが、Pythonの勉強もかねて一からプログラミングしてみました。
他者製品のような高度な機能はありませんが、シンプルなプログラミングなので、仕組みがわかりやすく、誰でも容易にカスタマイズが可能です。
本プログラムはTeMaKiプロジェクトとしてオープンソース化することにしました。

ちなみに、TeMaKiは「Text Markup Kit」の略です。


実行環境

最新版のPythonが必要です。こちらではMac上のPython3.7で動作確認を行なっています。


使い方

postsディレクトリにMarkDownファイルを入れてください。
python main.py allを実行すると、HTML生成が始まり、ブラウザーからhttp://0.0.0.0:8888/へアクセスすることで、生成されたコンテンツを確認することができるようになります。終了する場合はcontrol + cで終了してください。
python main.py all
All convert ...
http server start.... http://0.0.0.0:8888/
